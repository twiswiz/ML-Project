Project Overview:

This project is divided into five parts that cover employee attrition prediction, salary simulation, regression modeling, and loss estimation. It combines classification and regression techniques using machine learning models.
 Structure and Content
Part 1: Attrition Prediction
Data Import & Preprocessing: Loads and visualizes the HR dataset.

Modeling: Trains Logistic Regression and SVM classifiers.

Evaluation: Includes confusion matrix, ROC, and precision-recall curves for model comparison.

Part 2: Simulating Future Salaries
Projects salary growth based on performance metrics.

Uses visualization for better interpretation.

Part 3: Salary Prediction
Modeling: Applies Random Forest and XGBoost regressors.

Evaluation: Performance measured using R² and error metrics.

Comparison: Both models are compared for predictive power.

Part 4: Identifying “Likely to Stay” Employees
Selects employees less likely to leave using model predictions and probability thresholds.

Part 5: Estimate Expected Salary Loss
Combines attrition risk and salary data to estimate potential financial impact of attrition.

Ends with salary distribution plots for visual analysis.
